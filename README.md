# Bug Report Form

A simple bug reporting form.

**Live demo:** [https://beryl01.github.io/Bug-Report-Form/](https://beryl01.github.io/Bug-Report-Form/)

## Features

- Fill in a bug report with your name, a title, severity level, and description
- Validation on every field — clear error messages if something is missing or too short
- All saved reports are listed on the page each time you open it

## How to run locally

Open `index.html` directly in your browser — no server needed.

## How to make changes

1. Edit `index.ts`
2. Run `npx tsc` in the terminal to compile it to `index.js`
3. Refresh the browser
